      Manual

      Authors is the table name of all authors. Authors can be added to any content. Authors and content have a 0 to many relationship.

      Note: Please keep the authors unique, the code does not check for uniqueness itself.

      Field Values Required Notes
      id Number Yes Autofilled by tracker.
      name String Yes Format: Exact credited name, can include extra letters like: Greg A. Vaughan
      created_at DateTime Yes Autofilled by tracker
      updated_at DateTime Yes Autofilled by tracker
      deleted_at DateTime Yes Autofilled by tracker. If this is set then the content is officially 'deleted'. This is effectively a soft delete.
